להלן הפעולות לביצוע עדכון מסביבת פיתוח לטסט:
------------------------------------------------------------------------


1. מומלץ: למחוק גרסאות ישנות מה-SDE דרך משתמש sa.

2. מומלץ: להריץ כלי Rebuild Index על כל הטבלאות ב-SDE.

3.  לאתחל את הפרויקט בסביבת פיתוח עם הכלי לאתחול (Reinitialize)

4. לנקות היסטוריית הרצות: Analysis -> History ,בחירת כל הכלים שרצו ולחיצה על X.

5. לייצא את ה-tasks כקובץ בתיקיית Tasks

5. למחוק את כל התיקיות והקבצים בתיקיית TestEnvironment ולהעתיק קבצים חדשים מ-DevelopmentEnvironment.

6. בקובץ configs.py שהועתק- לשנות את המשתנה של הסביבה ל- "Test".

7. בפרויקט, לייבא את ה-tasks מחדש מתיקיית ה-tasks בטסט לאחר שהועתקו הקבצים.

8. בכל script-tool שיש ב-tool box של הפרויקט לעדכן את נתיב קובץ ההרצה (הסקריפט) לתיקיית הסקריפטים בטסט במקום התיקייה של הפיתוח.

9. לוודא שבאפשרויות של הפרויקט הנתיבים לתיקיות מוגדרים נכון לסביבת הטסט (Project->Options->Current Settings).

10. ךהעלות את הקבצים המעודכנים לגיטהאב.
