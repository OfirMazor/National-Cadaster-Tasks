Project Description

This project focuses on designing and implementing standardized, wizard-based workflows within ArcGIS Tasks. These step-by-step processes will be used to manage and execute cadastral changes and modifications directly within the Israeli National Cadaster dataset, which is maintained using ESRI's Parcel Fabric technology. The goal is to ensure consistency and efficiency in data updates.
